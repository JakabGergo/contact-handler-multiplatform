# Contact handler Multi-Platform App

## 🏗Backends
- Spring Boot Service: A modern, robust backend utilizing the Spring Framework for dependency injection and automated configuration.
- Simple Java Service: A lightweight, Java implementation following clean, multi-layer architecture patterns.

## 💻Client Interfaces
- Desktop Client: A native application interface for direct contact management from the OS.
- Web API (Servlets): A web-based interface utilizing Java Servlets to handle requests and manage data over HTTP.
